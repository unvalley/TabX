# TabX

TabX manages your tabs.

<img align="right" src=src/assets/tabX.png height="100px">

## Features

- ✅ Store your tabs
- 💎 Simple UI
- 🌌 Built-in Dark mode
- 🔍 Fuzzy Search (thanks to [Fuse.js](https://github.com/krisk/Fuse)! )
- ⛓ OneTab compatible
